\subsubsection{Word wrap - Continue code on next line}

You can use word wrap as default in Visual Studio Code settings by pressing \code{Ctrl + ,} and searching for 

\code{editor.wordWrap} 

Or you can change it for the current file by pressing \code{Ctrl + Shift + P} and searching for

\code{View: Toggle Word Wrap}

Or edit the shortcut key at 

\texttt{File \ra\ Preferences \ra\ Keyboard Shortcuts}
